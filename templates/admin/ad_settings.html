הגדרות נפוצות
  • Base DN: DC=domain,DC=com (החלף עם הדומיין שלך)
  • Bind DN: CN=ServiceAccount,CN=Users,DC=domain,DC=com
  • User Search Base: CN=Users,DC=domain,DC=com
  • User Search Filter: (sAMAccountName={username})
