חשוב: יש להוסיף כתובת זו ב-Azure Portal → App Registration → Authentication → Redirect URIs
שלבי הגדרה ב-Azure Portal:
  1. צור App Registration חדש או השתמש בקיים
  2. העתק את Tenant ID ו-Application (Client) ID מהעמוד הראשי
  3. ב-Certificates & secrets, צור Client Secret חדש והעתק את ה-Value
  4. ב-Authentication, הוסף את ה-Redirect URI המוצג למעלה
  5. ב-API permissions, ודא שיש הרשאה ל-User.Read (Microsoft Graph)
