איך להשתמש במדריך?

המלצות:
  1. משתמש חדש? התחל מ"התחברות למערכת" ואז עבור למדריך התפקיד שלך
  2. מחפש משהו ספציפי? השתמש ב"שאלות נפוצות" או חפש במדריך התפקיד
  3. רוצה להתקדם? קרא את "טיפים ועצות" לשימוש מתקדם
  4. יש בעיה? בדוק את "שאלות נפוצות" בקטגוריה "בעיות טכניות"
טיפ לחיפוש מהיר:

השתמש ב-Ctrl + F (או Cmd + F במק) כדי לחפש מילות מפתח בכל דף. כל המדריכים כוללים תוכן עניינים פנימי עם קישורים מהירים.
