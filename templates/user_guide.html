סקירה כללית

מערכת ניהול ועדות היא פלטפורמה ווב מתקדמת לארגון ותזמון ישיבות ועדות, ניהול אירועים וארגון חטיבות.

🎯 מטרת המערכת

ניהול יעיל של ועדות ארגוניות עם תזמון אוטומטי חכם, מעקב אחר אירועים, וחישוב מועדים אוטומטי תוך כיבוד SLA ואילוצים עסקיים.

✨ יתרונות המערכת
  • חיסכון בזמן תזמון
  • איזון עומסים אוטומטי
  • שקיפות ומעקב מלא
  • אינטגרציה עם מערכות ארגוניות

התחלת עבודה

התחברות למערכת

המערכת תומכת ב-3 שיטות אימות:

  1. אימות מקומי - שם משתמש וסיסמה
  2. Active Directory - משתמשים ארגוניים (LDAP)
  3. Azure AD / Microsoft 365 - כניסה עם חשבון Microsoft
ℹ️ משתמש ברירת מחדל: admin / admin123 (יש לשנות את הסיסמה בהתחברות הראשונה)

ממשק המשתמש הראשי

  • דף הבית - יומן אינטראקטיבי ודשבורד מידע
  • אירועים - ניהול אירועים בישיבות ועדה
  • תזמון אוטומטי - יצירה אוטומטית של לוח זמנים
  • הגדרות - ניהול חטיבות, מסלולים, סוגי ועדות ותאריכי חריגים
  • סל מחזור - שחזור פריטים מחוקים
  • יומן ביקורת - מעקב אחר שינויים (Admin בלבד)

תכונות עיקריות

ניהול ארגוני מתקדם

חטיבות ומסלולים: ארגון היררכי עם צבעים מותאמים אישית לזיהוי ויזואלי. כל חטיבה יכולה לכלול מספר מסלולים ולהגדיר סוגי ועדות משלה.

ניהול סוגי ועדות גמיש

ועדות שבועיות וחודשיות: יצירה דינמית של סוגי ועדות עם תזמון גמיש. ועדות חודשיות יכולות להיות מוגדרות לשבוע מסוים בחודש (ראשון, שני, שלישי, רביעי). תמיכה בועדות תפעוליות ללא קשר לאירועים.

תזמון אוטומטי חכם

יצירה אוטומטית של לוח זמנים: תזמון למספר חודשים בבת אחת עם אילוצים עסקיים - ועדה אחת ביום, כיבוד ימי עבודה (ראשון-חמישי), התחשבות בחגים ומגבלות שבועיות. ממשק ביקורת ואישור לפני יצירת ישיבות.

ניהול אירועים מתקדם

אירועים עם חישוב מועדים: יצירת אירועים הקשורים לישיבות ועדה ספציפיות, עם חישוב אוטומטי של מועדי שלבים (קול קורא, הגשה, סקירה, תגובה) תוך כיבוד SLA וימי עבודה. אימות אוטומטי של התאמת חטיבות.

יומן אינטראקטיבי

היומן האינטראקטיבי הוא הלב של המערכת - ממשק ויזואלי מתקדם המאפשר ניהול מהיר ויעיל.

תכונות היומן

  • צבעי חטיבות - כל חטיבה מוצגת בצבע ייחודי
  • תפריטי הקשר - לחיצה על תאריך ריק ליצירת ועדה, לחיצה על תאריך עם ועדה להוספת אירוע/עריכה/מחיקה
  • ניווט מהיר - מעבר בין חודשים עם חצים או בחירה ישירה
  • מקרא צבעים - הצגת כל החטיבות עם הצבעים שלהן
💡 טיפ: השתמש ביומן האינטראקטיבי ליצירה מהירה - זו הדרך המהירה ביותר ליצור ועדות ואירועים!

תזמון אוטומטי

מערכת התזמון האוטומטי יוצרת לוח זמנים חכם למספר חודשים בבת אחת.

תהליך התזמון

  1. בחירת פרמטרים: שנה, חודשים (אפשרות למספר חודשים), וחטיבות
  2. סקירת הצעות: עמוד ביקורת עם כל ההצעות ותוצאות אימות
  3. אישור ויצירה: בחירת ישיבות רצויות ויצירתן במערכת

אילוצי תזמון

  • ימי עסקים: ראשון עד חמישי בלבד
  • ועדה אחת ביום בכל המערכת
  • כל סוג ועדה ביום קבוע
  • התחשבות בחגים ותאריכי חריגים
  • מגבלות שבועיות על מספר ישיבות
  • ועדות חודשיות בשבוע מסוים

ניהול אירועים

יצירת אירוע

ישנן שתי דרכים ליצור אירוע:

  1. מהיומן (מומלץ): לחץ על תאריך עם ועדה קיימת → "הוסף אירוע לועדה"
  2. מעמוד האירועים: לחץ "צור אירוע חדש" ובחר ועדה ומסלול

חישוב מועדי שלבים

בעת יצירת אירוע, המערכת מחשבת אוטומטית 5 מועדים:

  • פרסום קול קורא - הזנה ידנית
  • סגירת קול קורא - מחושב אוטומטית
  • מועד הגשה - מחושב אוטומטית
  • מועד סקירה - מחושב אוטומטית
  • מועד תגובה - מחושב אוטומטית

כל המועדים מחושבים תוך כיבוד ימי עבודה, חגים, והגדרות SLA של המסלול.

סל מחזור

סל המחזור מאפשר שחזור של פריטים שנמחקו בטעות.

מחיקה רכה

כאשר אתה מוחק ועדה או אירוע:

  • הפריט מועבר לסל המחזור (לא נמחק לצמיתות)
  • מוסתר מהתצוגה הרגילה
  • ניתן לשחזור בכל עת
  • נרשם ביומן הביקורת

שחזור פריטים

  1. גש ל"סל מחזור" מהתפריט הראשי
  2. מצא את הפריט (ועדה או אירוע)
  3. לחץ "שחזר" ליד הפריט
  4. אשר את השחזור - הפריט יחזור למערכת
⚠️ אזהרה: מחיקה קבועה (Permanent Delete) היא בלתי הפיכה! שימוש בזהירות.

יומן ביקורת

הרשאות: Admin בלבד

מה נרשם?

כל פעולה במערכת מתועדת אוטומטית:

  • התחברות וניתוק (כולל כושלים)
  • יצירה, עריכה ומחיקה של כל הישויות
  • שחזור פריטים מסל המחזור
  • שינויים בהגדרות מערכת
  • ניהול משתמשים והרשאות

פרטים נרשמים

לכל רשומה: תאריך/שעה, משתמש, פעולה, ישות, מזהה, פרטים נוספים, IP Address, User Agent, סטטוס והודעת שגיאה (במקרה של כישלון).

יכולות

  • סינון מתקדם - לפי תאריך, משתמש, פעולה, ישות
  • סטטיסטיקות גרפיות - גרפי עמודות, עוגה וקווים
  • ייצוא CSV - להורדה ולניתוח חיצוני

סנכרון לוח שנה

הרשאות: Admin בלבד

המערכת יכולה לסנכרן אוטומטית ישיבות ועדות ומועדי אירועים ללוח השנה של Microsoft 365.

הפעלת הסנכרון

  1. גש ל"הגדרות" → "הגדרות מערכת"
  2. מצא את קטגוריית "סנכרון לוח שנה"
  3. הפעל סנכרון ושמור
  4. הזן כתובת אימייל Microsoft 365
  5. התחבר ואשר הרשאות ב-Microsoft

מה מסונכרן?

  • ישיבות ועדה (2 שעות)
  • מועד פרסום קול קורא
  • מועד סגירת קול קורא (עם התראה 3 ימים לפני)
  • מועד הגשת בקשות (עם התראה)
  • מועד תגובה למבקשים

הסנכרון רץ אוטומטית ברקע בתדירות מוגדרת (ברירת מחדל: כל שעה).

אימות משתמשים מתקדם

המערכת תומכת ב-3 שיטות אימות:

1. אימות מקומי (Local)

סיסמאות מאוחסנות במערכת. מתאים לסביבות קטנות ופיתוח. יצירת משתמשים ידנית על ידי Admin.

2. Active Directory (LDAP)

אימות ארגוני מול שרת AD. מתאים לארגונים עם תשתית AD קיימת. יצירת משתמשים אוטומטית בהתחברות ראשונה עם מיפוי קבוצות לתפקידים.

3. Azure AD / Microsoft 365 (OAuth 2.0)

אימות ענן מול Microsoft Azure. מתאים לארגונים עם Microsoft 365. תומך ב-SSO, MFA ו-Conditional Access.

ℹ️ הערה: המערכת תומכת במשתמשים מקומיים ו-AD/Azure במקביל. כל משתמש מסומן עם מקור האימות שלו.

המלצות חכמות

מערכת ההמלצות החכמה עוזרת למשתמשים לבחור את הועדה המתאימה ביותר בעת הוספת אירוע חדש.

קריטריוני הניקוד

המערכת מעריכה כל ועדה לפי:

  • זמינות מקום ⭐ - האם יש מקום נוסף? כמה בקשות כבר יש?
  • התאמת SLA 📅 - האם מועד הועדה מתאים למועדי SLA של המסלול?
  • עומס הועדה ⚖️ - ועדות פחות עמוסות מועדפות
  • טווח זמן 📆 - בונוס לועדות בטווח סביר
  • אילוצים שבועיים 🗓️ - קנס אם השבוע עמוס מדי

תצוגת ההמלצות

הועדות מוצגות עם:

  • כוכבים צבעוניים (1-5 כוכבים)
  • ניקוד מספרי (0-100)
  • סיבת ההמלצה - הסבר קצר
  • מיון אוטומטי - הועדות הטובות ביותר בראש
💡 טיפ: Admin יכול להתאים את פרמטרי ההמלצות (בונוסים, קנסים, טווחים) בדף "ניהול אילוצים".

שאלות נפוצות

שאלות כלליות

ש: איך אני מתחיל להשתמש במערכת?

ת: התחבר עם שם משתמש וסיסמה (או דרך AD/Azure). גש לדף הבית לראות את היומן ולהתחיל ליצור ועדות ואירועים.

ש: מה ההבדל בין סוג ועדה לישיבת ועדה?

ת: סוג ועדה הוא הגדרה כללית (כמו "ועדת הזנק") עם כללי תזמון. ישיבת ועדה היא מופע ספציפי (ועדת הזנק ב-15/01/2025).

ש: כמה ישיבות יכולות להתקיים ביום אחד?

ת: מקסימום ישיבה אחת ביום בכל המערכת - זהו כלל יסוד שמבטיח איזון עומסים.

ש: איך משחזרים פריט שנמחק בטעות?

ת: גש ל"סל מחזור" מהתפריט הראשי, מצא את הפריט, ולחץ "שחזר". הפריט יחזור למערכת.

ש: איך מפעילים סנכרון לוח שנה עם Microsoft 365?

ת: Admin בלבד - גש ל"הגדרות" → "הגדרות מערכת", הפעל את "סנכרון לוח שנה", הזן כתובת אימייל Microsoft 365, והתחבר לאישור הרשאות.

ש: איך רואים מי שינה משהו במערכת?

ת: Admin בלבד - גש ל"יומן ביקורת" מתפריט המשתמש. שם תוכל לראות את כל הפעולות, מי ביצע, מתי, ועל איזה פריט.

ש: איך עובדת מערכת ההמלצות לועדות?

ת: כשאתה יוצר אירוע חדש, המערכת אוטומטית ממליצה על ועדות מתאימות לפי זמינות, התאמת SLA, עומס ועוד. הועדות מדורגות עם כוכבים וניקוד.

ש: איך המערכת מחשבת את מועדי השלבים?

ת: אתה מזין תאריך פרסום קול קורא, והמערכת מחשבת אוטומטית את כל המועדים הבאים (סגירת קול קורא, הגשה, סקירה, תגובה) לפי הגדרות ה-SLA של המסלול, תוך כיבוד ימי עבודה וחגים.

ש: מה ההבדל בין Editor ל-Viewer?

ת: Viewer יכול רק לצפות בנתונים. Editor יכול גם ליצור ולערוך אירועים בחטיבות שלו (כשתקופת העריכה פתוחה). Admin יכול הכל.

טיפים לשימוש יעיל

  • השתמש ביומן האינטראקטיבי - הדרך המהירה ביותר ליצירת ועדות ואירועים
  • תכנן עם התזמון האוטומטי - יצור לוח זמנים לכמה חודשים בבת אחת
  • הגדר סוגי ועדות מראש - לפני השימוש בתזמון האוטומטי
  • נהל תאריכי חריגים - עדכן חגים בתחילת השנה
  • השתמש בצבעי חטיבות - לזיהוי מהיר ביומן
  • בדוק הרשאות - וודא שיש לך הרשאות מתאימות לפעולות שאתה רוצה לבצע
